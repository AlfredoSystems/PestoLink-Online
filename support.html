PestoLink Support

PestoLink is a Bluetooth controller app for AlfredoSystems robots. Use it to connect your phone or tablet to a compatible robot over BLE and send joystick and button commands in real time.

Frequently Asked Questions

My device won't connect to the robot.
Make sure Bluetooth is enabled on your device and the robot is powered on. Tap the 🔗 button to scan. If the connection fails, try refreshing the app with the 🔄 button and reconnecting.

The app doesn't find my robot.
Ensure your robot is running compatible PestoLink firmware. Only robots advertising the PestoLink BLE service will appear in the device list.

Bluetooth isn't working on Safari.
Safari does not support Web Bluetooth. Use the PestoLink iOS app instead of the web version.

The joystick or buttons aren't responding.
Check that you're in Mobile Layout mode (toggle in the interface panel). If using a gamepad, make sure it is connected and selected via the 🎮 button.
